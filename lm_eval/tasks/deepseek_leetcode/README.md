# DeepSeek LeetCode

## Paper

Title: DeepSeek-Coder: When the Large Language Model Meets Programming

Abstract: https://arxiv.org/abs/2401.14196

DeepSeek LeetCode is a benchmark derived from LeetCode problems for evaluating code generation capabilities. It tests models on competitive programming-style problems.

Homepage: https://huggingface.co/datasets/davidheineman/deepseek-leetcode
